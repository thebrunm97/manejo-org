# Research Phase (Context Discovery)

Passo 1: Leia os arquivos Python do diretório `pmo_bot/` relevantes para a funcionalidade solicitada.
Passo 2: Leia os arquivos Go do diretório `pmo-bot-go/` para entender o estado atual da refatoração.
Passo 3: Mapeie as discrepâncias (o que o Python fazia que o Go ainda não faz, como lógica N:N de canteiros, etc).
Passo 4: Gere um documento mental ou resumo no chat chamado "Context Summary" e pergunte ao usuário: "Posso prosseguir com o /plan?"
